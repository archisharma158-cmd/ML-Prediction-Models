# ❤️ Heart Disease Prediction

## 📌 Project Overview

This project focuses on predicting the likelihood of heart disease using machine learning classification algorithms. The workflow includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and performance evaluation to identify the most effective predictive model.

## 🤖 Models Used

- Logistic Regression
- Decision Tree Classifier
- Random Forest Classifier

## 🏆 Best Performing Model

Both **Logistic Regression** and **Random Forest Classifier** achieved the highest accuracy, making them the best-performing models for this dataset.

## 📂 Dataset

- **Dataset:** Heart Disease Dataset
- **Source:** Kaggle
